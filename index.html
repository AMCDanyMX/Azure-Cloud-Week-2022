Acerca de este evento

Azure Cloud Week es un evento el cual tiene como objetivo tener una secuencia de ponencias y workshops acerca de la nube de Microsoft, Azure, de sus servicios y de las aplicaciones que trabajan en conjunto con ella.

Cabe mencionar, que hoy en día, Microsoft Azure, se ha convertido en una de las nubes informáticas mas cotizadas y utilizadas a nivel mundial, empleada por diversas compañías para agilizar el trabajo empresarial en diversas áreas, que cada vez se va haciendo más esencial el uso de las nubes.

Por eso, al ser una de las nubes que cuenta con diversos servicios que proporcionan una facilidad en cuanto al cómputo en la nube, hacen de Azure un asunto del que se pueden extraer muchas cosas de las que se pueden hablar y poner en práctica.

Por lo tanto, en este evento se ambiciona que se concentren personas que presenten como se pueden implementar los servicios de Azure actualmente. Con esto, se busca instruir a la comunidad en general acerca de las formas en las que se puede utilizar Azure para obtener una comprensión y relación amplia de la misma.

Por otro lado, se trata de tener charlas prácticas y teóricas que demuestren el uso y la importancia que tiene Azure para múltiples propósitos, proyectos y soluciones.

Internet de las cosas

El Internet de las cosas (IoT) es el proceso que permite conectar elementos físicos cotidianos al Internet: desde objetos domésticos comunes, como las bombillas de luz, hasta recursos para la atención de la salud, como los dispositivos médicos.

Desarrollo web

Desarrollo web significa construir y mantener sitios web; es el trabajo que tiene lugar en un segundo plano y que permite que una web tenga una apariencia impecable, un funcionamiento rápido y un buen desempeño para permitir la mejor experiencia de usuario.

Seguridad en la nube

Proteja datos, aplicaciones e infraestructura rápidamente con servicios de seguridad integrados, que incluyen inteligencia de seguridad inigualable para ayudar a identificar anticipadamente amenazas que evolucionan con rapidez, de forma que pueda responder a ellas rápidamente.

phonelink

Desarrollo de Aplicaciones

El objetivo principal del desarrollo de aplicaciones es crear softwares que sirvan de herramienta para usuarios que utilizan aparatos portátiles de diferentes sistemas operativos. Existen tres tipos de desarrollo de aplicaciones: el desarrollo web, el desarrollo de aplicaciones móviles y desarrollo de aplicaciones multiplataforma o híbridas.

Ciencia de Datos

La ciencia de datos combina múltiples campos, como las estadísticas, los métodos científicos, la inteligencia artificial (IA) y el análisis de datos para extraer el valor de los datos. La ciencia de datos abarca la preparación de los datos para el análisis, incluida la limpieza, la agregación y la manipulación de los datos para realizar análisis avanzados.

Inteligencia Artificial

En términos simples, inteligencia artificial (IA) se refiere a sistemas o máquinas que imitan la inteligencia humana para realizar tareas y pueden mejorar iterativamente a partir de la información que recopilan. La IA se manifiesta de varias formas. Algunos ejemplos son:

Los chatbots, los asistentes inteligentes y los motores de recomendación

Machine Learning

Machine learning es una forma de la IA que permite a un sistema aprender de los datos en lugar de aprender mediante la programación Explícita. Conforme el algoritmo ingiere datos de entrenamiento, es posible producir modelos más precisos basados en datos. Un modelo de machine learning es la salida de información que se genera cuando entrena su algoritmo de machine learning con datos.

insights

Analítica

La analítica empresarial es el proceso de detectar, interpretar y comunicar patrones significativos en los datos, así como usar herramientas para que toda la organización pueda realizar cualquier pregunta sobre cualquier información en todos los entornos y dispositivos posibles.

Bases de Datos

Una base de datos es una recopilación organizada de información o datos estructurados, que normalmente se almacena de forma electrónica en un sistema informático. Normalmente, una base de datos está controlada por un sistema de gestión de bases de datos (DBMS). En conjunto, los datos y el DBMS, junto con las aplicaciones asociadas a ellos, reciben el nombre de sistema de bases de datos, abreviado simplemente base de datos.

Microsoft Office 365

Office 365 es la suite de Microsoft que nos permite trabajar con las herramientas de Office de toda la vida y las herramientas de colaboración más actuales, desde cualquier dispositivo y en cualquier lugar del mundo.

Microsoft Power Platform

Power Platform es un conjunto de herramientas de Microsoft integrada dentro de Office 365, preparada para analizar, crear soluciones y automatitzar los procesos de su empresa. Una poderosa plataforma que junta tres soluciones empresariales de Microsoft: Power BI, Power Apps y Power Automate.

Microsoft Azure

La plataforma Azure está compuesta por más de 200 productos y servicios en la nube diseñados para ayudarle a dar vida a nuevas soluciones que permitan resolver las dificultades actuales y crear el futuro. Puedes crear, ejecutar y administrar aplicaciones en varias nubes, en el entorno local y en el perímetro, con las herramientas y los marcos que prefiera.

Learn Student Ambassadors

La comunidad de Microsoft Learn Student Ambassadors es para estudiantes, como usted, que desean usar la tecnología para resolver problemas del mundo real con compañeros de ideas afines, establecerse como mentores y líderes en su comunidad y amplificar su impacto.

Los estudiantes embajadores tienen la oportunidad de conectarse con otros estudiantes de todo el mundo (¡de 102 países y contando!), fomentar sus propias comunidades en línea, obtener capacitación y obtener certificaciones, y enfrentar desafíos prácticos, todo mientras desarrollan habilidades técnicas y sociales clave para ayudarlos a triunfar. También pueden lograr un progreso significativo en los temas socialmente significativos que les preocupan.

Preguntas comunes

¿Qué es Microsoft Learn Student Ambassadors?

El programa Student Ambassadors de Microsoft Learn es un grupo global de embajadores del campus que están deseando ayudar a los estudiantes y sus comunidades, llevar la iniciativa en sus círculos tecnológicos locales y desarrollar aptitudes técnicas y profesionales para el futuro. Como miembro de Student Ambassadors, puede que llegue a organizar un hackathon para el campus o la comunidad, formar a otros estudiantes o participar como voluntario en una clase de STEM de educación primaria.

¿Qué es Microsoft Azure?

Azure es un conjunto de servicios en la nube de Microsoft. Con Azure es posible almacenar información y crear, administrar e implementar aplicaciones en cloud. Para utilizar Azure es necesario el pago de una cuota que recoge los servicios contratados(Pago por uso). Desde el portal de Microsoft Azure se puede acceder a diferentes servicios de infraestructura y plataforma para contratar aquellos que sean necesarios para la empresa o proyecto. En apenas unos clics es posible disponer de Microsoft Azure funcionando y listo para trasladar el trabajo a la nube.

¿Qué es Azure Cloud Week?

Azure Cloud Week es un evento organizado por estudiantes embajadores, el cual tiene como objetivo tener una secuencia de ponencias y workshops acerca de la nube de Microsoft, Azure, de sus servicios y de las aplicaciones que trabajan en conjunto con ella.
